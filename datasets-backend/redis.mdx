#Redis as a dataset

Redis is a great low-latency dataset for both metadata and vector retrieval.

## Set up your redis connection string for insertion
```python Python
import relevanceai as rai
rai.login()
rai.set_key("redis", "....")
rai.set_key("redis_connection_string", "....")
```

## Insert your data as normal
```python Python
ds = rai.Dataset(
    id = "hello_world"
)
ds.insert(
    documents=[
        {"_id": "doc1", "text": "tiger", "category": "land"},
        {"_id": "doc2", "text": "lion", "category": "land"},
        {"_id": "doc3", "text": "tigr", "category": "land"},
        {"_id": "doc4", "text": "car", "category": "land"},
    ]
)
```

## Unset your redis connection string
Unset it so your dataset isn't always inserting to Redis
```python Python
import relevanceai as rai
rai.delete_key("redis")
```
