title: "List and run custom step"
---

## List all the custom steps

```python Python 
import relevanceai as rai
rai.list_all_steps(custom_only=True)
```
example of output:
```
[{'id': 'Run_GPT2_gpt2_post',
  'name': 'Run GPT2',
  'description': 'Run GPT2',
  'input_schema': ['prompt'],
  'output_schema': ['results'],
  'required': ['prompt']}]
```

Run the custom step:
```python Python
from relevanceai.steps import RunStep

step = RunStep("Run_GPT2_gpt2_post", prompt="Hello")
```
