# Vector Search with Redis
Login and set your redis connection string you can also set this in the dashboard
```python Python
import relevanceai as rai
rai.login()
rai.set_key("redis-external", YOUR_REDIS_CONNECTION_STRING)
```
Its ready for search
```python Python
from relevanceai.steps import RedisSearch

results = RedisSearch(
    index="mlopscommunity",
    query="hello",
    vector_field="vector",
    model="text-embedding-ada-002",
    page_size=5
).run()
```
