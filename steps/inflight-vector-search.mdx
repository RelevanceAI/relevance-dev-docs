# Vector Search Inflight
Login and set your redis connection string you can also set this in the dashboard
```python Python
import relevanceai as rai
rai.login()
```
Its ready for search
```python Python
from relevanceai.steps import VectorizeAndSearchArray

results = VectorizeAndSearchArray(
    array=["car", "dog", "horse", "milk", "face"],
    query="vroom",
    page_size=1
).run()
```
