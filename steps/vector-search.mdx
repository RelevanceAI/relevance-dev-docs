Login and set your redis connection string you can also set this in the dashboard

```python Python
import relevanceai as rai

```
Its ready for search
```python Python
from relevanceai.steps import VectorSimilaritySearch

results = VectorSimilaritySearch(
    query="Hello",
    vector_field="text_vector_",
    dataset_id="hello_world"
    page_size=2,
)
```
